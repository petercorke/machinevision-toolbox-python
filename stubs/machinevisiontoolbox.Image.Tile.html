classmethod Image.Tile(tiles, columns=4, sep=2, bgcolor=None)ÔÉÅ

Tile images into a grid

Parameters:

  • tiles (iterable of Image) ‚Äì images to tile

  • columns (int, optional) ‚Äì number of columns in the grid, defaults to 4

  • sep (int, optional) ‚Äì separation between images, defaults to 1

  • bgcolor (scalar, string, array_like, optional) ‚Äì color of background, seen in the separation between images, defaults to black

Raises:

  • ValueError ‚Äì all images must have the same size

  • ValueError ‚Äì all images must have the same dtype

Returns:

grid of images

Return type:

Image instance
