machinevisiontoolbox.Image.warpÔÉÅ

Image.warp(U, V, interp=None, domain=None)ÔÉÅ

Image warping

Parameters:

  • U (ndarray(Wo,Ho)) ‚Äì u-coordinate array for output image

  • V (ndarray(Wo,Ho)) ‚Äì u-coordinate array for output image

  • interp (str, optional) ‚Äì interpolation mode, defaults to None

  • domain ((ndarray(H,W), ndarray(H,W)), optional) ‚Äì domain of output image, defaults to None

Returns:

warped image

Return type:

Image

Compute an image by warping the input image. The output image is \(H_o \times W_o\) and output pixel (u,v) is interpolated from the input image coordinate (U[u,v], V[u,v]):

\[Y_{u,v} = X_{u^\prime, v^\prime} \mbox{, where } u^\prime = U_{u,v}, v^\prime = V_{u,v}\]
Note:

Uses OpenCV.
