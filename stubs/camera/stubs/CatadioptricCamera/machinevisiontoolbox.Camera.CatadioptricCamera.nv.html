property CatadioptricCamera.nv

Get image plane height (base method)

Returns:

height

Return type:

int

Number of pixels in the v-direction (height)

Example:

>>> from machinevisiontoolbox import CentralCamera
>>> camera = CentralCamera.Default();
>>> camera.nv
1000
