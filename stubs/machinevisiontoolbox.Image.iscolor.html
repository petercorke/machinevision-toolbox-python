machinevisiontoolbox.Image.iscolor

property Image.iscolor

Image has color pixels?

Returns:

Image is color

Return type:

bool

Returns:

number of image dimensions

Return type:

int

Example:

>>> from machinevisiontoolbox import Image
>>> img = Image.Read('flowers1.png')
>>> img.iscolor
True
>>> img = Image.Read('street.png')
>>> img.iscolor
False
