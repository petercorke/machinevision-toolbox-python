Image.print(fmt: str | None = None, separator: str = ' ', precision: int = 2) → None[source]ÔÉÅ

Print image pixels in compact format

Parameters:

  • fmt (str, optional) ‚Äì format string, defaults to None

  • separator (str, optional) ‚Äì value separator, defaults to single space

  • precision (int, optional) ‚Äì precision for floating point pixel values, defaults to 2

Very compact display of pixel numerical values in grid layout.

Example:

>>> img = Image.Squares(1, 10)
